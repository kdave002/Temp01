# PROJECT_ROADMAP

## Product
- Name: DriftShield
- ICP: US B2B SaaS (100-800 employees), dbt + BigQuery users
- Core Promise: Prevent downstream breakage from schema drift by auto-generating safe repair PRs.

## Milestones
1. M0 Foundation (Day 1)
   - Repo scaffold, architecture docs, core logs
2. M1 Drift Detection (Week 1)
   - Schema diff engine + impact scoring
3. M2 Repair Generation (Week 2)
   - LLM-assisted semantic mapping + SQL patch draft
4. M3 Validation + PR Automation (Week 3)
   - Rule checks + GitHub PR formatter
5. M4 Pilot Pack (Week 4)
   - onboarding checklist, pricing, and pilot success metrics

## Human Checkpoints
- Checkpoint A: M1 complete
- Checkpoint B: M2 complete
- Checkpoint C: MVP pilot-ready
